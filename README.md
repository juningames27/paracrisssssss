# Quiz Namorada

Versão simples, sem banco de dados e sem API.

## Como usar na Vercel

- Suba estes arquivos no GitHub.
- Na Vercel, deixe o projeto como estático.
- A pasta `public` contém o site.
- Não precisa de `DATABASE_URL`, PostgreSQL, Render ou servidor Node.

Quando a pessoa escolher uma surpresa, o mimo abre direto na tela.
